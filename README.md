# Product Sales Analysis using AWS

## Project Overview
This project analyzes Product sales details using AWS cloud services.

## Tools Used
-> AWS S3
-> AWS Glue
-> Athena
-> SQL
-> Excel

## Architecture
CSV -> AWS S3 -> AWS Glue (creating a crawler) -> Athena -> SQL Queries

## Dataset
The dataset contains 180,000+ records of sales information including Product, Quantity_Ordered, Price, Order_Date and	Time.

## Analysis Performed
-> Total Revenue
-> Revenue by City
-> Top Selling Products
-> Best Sales Month
-> Peak Sales Hours
